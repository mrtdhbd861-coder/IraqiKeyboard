# كيبورد عراقي - مشروع Android Studio

## المحتوى
- لوحة حروف عربية (٣ صفوف)
- لوحة أرقام بشكل Numpad (١٢٣ / ٤٥٦ / ٧٨٩ / .٠)
- زر "تفكيك الكلمة": لما يكون مفعّل، كل حرف تكتبه ينحط بعده مسافة تلقائياً
  مثال: كتابة "العراق" تطلع "ا ل ع ر ا ق"
- زر "تدبيل": لما يكون مفعّل، أي رقم تكتبه وتضغط مسافة بعده ينكتب مرتين
  مثال: كتابة "45" ثم مسافة تطلع "45 45"

## كيفية البناء (على جهاز كمبيوتر فيه Android Studio)
1. نزّل Android Studio (من موقع developer.android.com/studio) اذا ما عندك.
2. افتح Android Studio -> Open -> اختر مجلد `IraqiKeyboard` هذا كامل.
3. خل يسوي Gradle Sync (يصير تلقائي، ينزل المكتبات المطلوبة - يحتاج انترنت أول مرة).
4. من الأعلى: Build -> Build Bundle(s)/APK(s) -> Build APK(s).
5. راح يطلعلك اشعار فيه "locate" اضغط عليه يوديك لمكان ملف الـ APK
   (عادةً: app/build/outputs/apk/debug/app-debug.apk)
6. انقل ملف الـ APK لموبايلك (عن طريق كابل، تيليكرام لنفسك، جوجل درايف...) ونصبه.

## كيفية التفعيل على الموبايل
1. افتح تطبيق "كيبورد عراقي" اللي نصبته.
2. اضغط "فتح إعدادات لوحات المفاتيح".
3. فعّل "كيبورد عراقي" من القائمة (راح يجيك تحذير عادي من أندرويد عن الصلاحيات - طبيعي لأي كيبورد).
4. ارجع لأي تطبيق (تلكرام مثلاً)، اضغط مطولاً على شريط المسافة بالكيبورد الحالي أو زر تبديل الكيبورد، واختر "كيبورد عراقي".

## تعديل شكل الحروف أو الأرقام
كل شي موجود بملف:
`app/src/main/java/com/iraqi/customkeyboard/IraqiKeyboardService.kt`

- `lettersRows` -> ترتيب حروف لوحة الحروف
- `numpadRows` -> ترتيب أزرار لوحة الأرقام
- منطق "تفكيك الكلمة" و"تدبيل" موجود بدوال `onKeyPress` و `onSpace`

## ملاحظة مهمة
هذا المشروع ما يبنى من Termux مباشرة (يحتاج Android Studio على كمبيوتر عادي).
اذا تريد بس تجرب الفكرة بسرعة من دون بناء APK، اطلبها وأسوي محاكاة بسيطة أو
نشوف طريقة نبني بيها APK جاهز نبعثه لك مباشرة.
